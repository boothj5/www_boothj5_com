Profanity

Profanity is a simple Jabber client written in C using ncurses and libstrophe, inspired by Irssi.

Intructions to install can be found on the Github Wiki

Screenshots

Click an image to load the fullsize version.

Features

  • XMPP chat, Jabber, GTalk
  • Desktop notifications
  • Typing notifications
  • Chat logging
  • Ability to send Tiny URLs
  • Customisable UI
  • Tab autocompletion
