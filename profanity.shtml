Profanity

Profanity is a simple Jabber client written in C using ncurses and libstrophe, inspired by Irssi.

Screenshots

Click an image to load the fullsize version.

Features

  • XMPP chat, Jabber, GTalk
  • Disable TLS for servers that don't support it
  • Console window for notifications
  • Up to 9 active chat windows
  • Customisation with preferences file
  • Autcomplete names of online contacts
  • Logging XMPP activity to file
