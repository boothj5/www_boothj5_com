Profanity

Profanity is a simple Jabber client written in C using ncurses and libstrophe, inspired by Irssi.

Screenshots

Click an image to load the fullsize version.

Features

  • XMPP chat
  • Enable/disable TLS
  • Console window for logging
  • Up to 9 active chat windows
  • Sending of random control characters to recipient on window resize
